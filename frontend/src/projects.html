Personal Website.

The animation on the homepage isn't a GIF! Every frame is actually rendered live, and written as text directly to the page. I've always found ASCII art really cool, so this was a fun little project. The logic to compute the Perlin noise and render it into ASCII is written in Rust, and compiled into WASM. I haven't worked with WASM before, and so this was a fun experiment! Getting it to run at a high enough frame-rate was an interesting challenge. You can check out the source code here.

RAFT Consensus.

As part of a module at university, I built an implementation of the RAFT distributed consensus algorithm, in Elixir. Working with Elixir was fun, and trying to reason about the correctness of a distributed algorithm was interesting.

WACC Compiler.

As part of my compiler's course, I built a compiler for WACC, a while-like imperative language. I worked with parser-combinators, which I really enjoyed. Programming in Scala was really fun too.
